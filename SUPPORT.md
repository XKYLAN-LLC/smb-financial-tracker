# Support

Use GitHub issues for public bugs, documentation gaps, and feature ideas.

Do not include private financial records, source documents, account numbers, tax IDs, addresses, emails, phone numbers, screenshots, statements, receipts, or application IDs in a public issue.

## Where To File

- Bug: use the bug report template.
- Documentation gap: use the documentation improvement template.
- Product idea: use the feature request template.
- Privacy or security concern: use the privacy or security template if no sensitive data is included. If sensitive data is involved, follow `SECURITY.md` and do not post it publicly.

## Support Boundary

This project can help organize records, preserve source context, calculate summaries, and prepare reviewable handoffs. It is not a substitute for professional tax, legal, benefits, insurance, accounting, or financial advice.
